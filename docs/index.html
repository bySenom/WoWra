🛡️ BuffReminder WoW

Ein WeakAura-ähnliches externes Overlay-Tool für World of Warcraft – mit Buff-Timern, Profilen, TTS und WoW-Fenster-Tracking.

⬇ Download 📂 GitHub Repository

Features

🎯

WoW-Fenster Tracking

Das Overlay heftet sich automatisch an dein WoW-Fenster und folgt ihm bei Bewegung. Versteckt sich bei Alt-Tab oder Minimierung.

📋

Profile

Erstelle verschiedene Buff-Profile für verschiedene Klassen oder Situationen. Schneller Wechsel per Hotkey.

⏱️

Visueller Countdown

Großer, zentrierter Countdown über dem WoW-Fenster in den letzten 5 Sekunden – WeakAura-Style.

🔊

Individuelle Sounds

Eigene .wav-Sounddateien pro Buff oder der Standard-Beep. Plus TTS-Sprachansage über Windows SAPI.

🔗

Buff-Abhängigkeiten

Buff A startet nur, wenn Buff B aktiv ist. Ideal für Abhängigkeiten wie Lifebloom → Rejuvenation.

⏰

Früh-Alarm

Alarm X Sekunden vor Ablauf – damit du nie zu spät refreshst. Konfigurierbar pro Buff.

🖱️

Maustasten-Support

Unterstützt Maustasten (M3–M5) als Hotkeys via GetAsyncKeyState-Polling – zuverlässig auch in WoW.

🎨

Dark-Theme Overlay

Transparentes, minimalistisches Overlay mit farbigen Countdown-Balken (Grün → Gelb → Rot).

🔄

Auto-Updater

Prüft beim Start automatisch auf neue Versionen via GitHub Releases.

So funktioniert's

1

Starten

Starte main.py als Administrator. Das Overlay erscheint und heftet sich automatisch an WoW.

2

Buffs konfigurieren

Klicke ⚙ im Overlay. Füge Buffs hinzu, weise Hotkeys zu, konfiguriere Dauer, Sound und Abhängigkeiten.

3

Hotkey drücken

Drücke deinen Hotkey (z.B. F5 oder M5) wenn du den Buff castest. Der Timer startet – erneutes Drücken stackt die Dauer.

4

Erinnerung erhalten

Sound + TTS + visueller Countdown erinnern dich rechtzeitig – nie wieder einen Buff vergessen!

Installation

# Repository klonen
git clone https://github.com/BuffReminderWoW/BuffReminderWoW.git
cd BuffReminderWoW

# Abhängigkeiten installieren
pip install -r requirements.txt

# Als Administrator starten
python main.py
⚠ Hinweis: Das Programm muss als Administrator gestartet werden, damit die globalen Hotkeys auch innerhalb von WoW funktionieren. Windows 10/11 erforderlich.

Konfiguration

Jeder Buff kann individuell konfiguriert werden – entweder über das ⚙ Einstellungs-Fenster oder direkt in der config.json.

Feld Beschreibung
name Buff-Name (wird im Overlay angezeigt)
hotkey Taste zum Aktivieren (z.B. F5, M5, ctrl+1)
duration Sekunden die pro Tastendruck addiert werden
max_duration Maximale Gesamtdauer des Timers
alert_before Alarm X Sekunden vor Ablauf (0 = bei Ablauf)
sound Standard-Beep bei Ablauf (true/false)
sound_file Eigene .wav-Datei statt Standard-Beep (Pfad)
tts Sprachansage bei Ablauf (true/false)
depends_on Name eines Buffs der aktiv sein muss, damit dieser startet
